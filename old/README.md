# arthur1.github.io

## init

```shell
git clone git@github.com:Arthur1/arthur1.github.io.git
cd arthur1.github.io
nodenv install
yarn
```

## development server

```shell
yarn start
```

## build

Artifacts will be written out under `/public`.

```sh
yarn build
```

## deploy

Don't forget to execute `yarn build` before deployment.

```sh
yarn deploy
```
